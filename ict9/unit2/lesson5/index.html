Overview

In this lesson we’ll add some classic platformer mechanics to our game:

  • Pickups — coins, gems, or hearts that disappear when collected.
  • HUD — a display that will always show our updated score
  • Stomp-to-Defeat — jumping on enemies defeats them, while side hits damage the player.

Pickups

Pickups are collectibles that make the game more engaging and rewarding. They give players a reason to explore levels and often tie into the scoring or health system. In this step, we’ll create a simple pickup (like a coin or gem), animate it so it looks alive, and make it disappear when the player touches it. Later, we’ll connect pickups to the HUD to track score and lives.

  1. First lets add the Ellipse movement extension to make our pickup appear to be floating
    • Open Project Manager → Extensions
    • Click on the "+" next to Extensions and search for the EllipseMovement extension. Sellect Install in project
  2. Add a new Sprite and rename it jewel_pickup. Import yellowJewel.png file from assets → PNG → Items
  3. Search and add the EllipseMovementbehaviour.
  4. Edit the following settings:
    • Radius of the movement on X axis: 0
    • Radius of the movement on Y axis : 8 - how much the object moves up and down
    • Loop duration : 1 - how long it will take to move up and down in seconds
  5. Add some pickups to your scene and preview your work. You should see a smooth, perpetual float.

2. HUD

Now we want to display a score and add to it each time the Player collides with a Pickup. This display will always be visible no matter where the player goes on the screen. It is called a Heads Up Display.

  1. Open the Layers Panel and Add a Layer. Name the layer HUD.

    HUD stands for Heads Up Display

  2. Ensure the HUD is selected and Add Object selecting BBText
  3. Add the following properties
    • Object name : Score
    • BBCode text : Score : 0 make not of the spacing
    • Base color : [leave as black or choose a colour you like]
    • Font : [choose any font you want]
  4. Select Apply
  5. Add the Score object to your scene in the top left corner.
  6. Preview your game and you should see your Score text stationary in the corner.

Now that we have everything set up, we can now add events that will:

  • Add the score variable to the HUD
  • Update the score when the player collides with a pickup

A scene variable called Score will keep track of the number of jewels the player collides with.

  1. Open the Project Manager
  2. Right-click on the three dots ... next to Level 1 under Scene
  3. Add a variable called score with a number value of 0
  4. Add action in the event with the blank condition at the top
    • Score → BBCode text = "Score : " + score

        Add spaces in between the quotes

        After the + start typing "score" and select it from the menu - or you will create a new variable.

  5. Add new event
    • Condition : Player → Collision → Object → jewel_pickup
    • Action : jewel_pickup → Objects → Delete the object
    • Action : Other actions → Variables → Change variable value
      • Variable : score
      • Modifiation sign : +(add)
      • Value : 1
  6. Save and Preview your game - your player should be able to collect jewels and the score increases.

3. Stomp-to-Defeat

Now we want to add events that will kill the enemy by bouncing on it AND cause damage to the player if the player runs into it. This we call "Stomp-to-Defeat"

We want to apply the following logic:

  • If the Player jumps onto of the Enemy, the enemy is killed and the Player bounces off the Enemy
  • If the Player runs into the enemy from the side, they flash and get hurt.

    • 1. Stomp to bounce (top hit)

      Add a new event

      • Conditions :
        • Player → Collision with → Enemy
        • Player → Is Falling
        • Other conditions → Advanced → Events and control flow → Compare two numbers
          • First Expression : Player.Y()
          • Sign of the test : < (less than)
          • Second Expression : Enemy.Y()

            • Remember: You should select from the list when given the option

      • Actions :
        • Enemy → Delete the object
        • Player → Allow jumping again
        • Player → Simulate jump key press

        Save your work and test out your code.

    2. Player Damage (side/below hit)

      Add a new event

      • Conditions :
        • Player → Collision → Enemy
        • Player.Y() >= Enemy.Y() Copy and paste the previous code and change the sign
      • Actions :
        • Player → Opacity → Set to = 50
        • Other actions → Timers and time → Wait X seconds → 0.06
        • Player → Opacity → Set to = 255

Now you have a Player that kills the enemy when you bounce on it, and flashes when it runs into it.
