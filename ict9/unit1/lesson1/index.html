How to save the file (H Drive, correct folder)
  1. Open the download in Word.
  2. Go to File → Save As → choose your H Drive.
  3. Create a folder named U1 Essential Skills (if not already created) in your ICT 9 folder.
  4. Save the file as 1_1_Scavenger_Hunt_YourName.docx.
How to export as PDF (Word)
  1. In Word, go to File → Save As (or Export).
  2. Choose file type PDF (.pdf).
  3. Save to the same folder: ICT_9/1_1_Scavenger_Hunt_YourName.pdf.
How to submit to MS Teams
  1. Open the 1.1 Twenty Questions About Me assignment in MS Teams.
  2. Attach both files:
    • 1_1_Twenty_Questions_About_Me.docx
    • 1_1_Scavenger_Hunt_YourName.pdf
  3. Click Turn in.
