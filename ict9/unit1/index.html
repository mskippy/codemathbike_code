Welcome

Unit 1: Essential Skills sets you up for success in ICT 9. We’ll build habits for file management, submissions in MS Teams, typing fluency, and everyday digital literacy that we’ll apply across all projects.

What You’ll Learn

  • Create and organize folders in OneDrive and Home Drive (and keep them tidy).
  • Work with common file types: DOCX, PDF, PNG/JPG, ZIP.
  • Download, rename, move, and submit files correctly in MS Teams.
  • Type with accuracy and speed using short daily warm-ups.
  • Apply core digital literacy (passwords, online identity, fair use, AI use, etc.).

How We’ll Do It

  • 1.1 Welcome & Scavenger Hunt — team up, find things, set up your folders.
  • Typing Routine — 10-15 min at the start of most classes.
  • Dig Lit Fridays — short, practical mini-lessons (twice a week).
  • Ongoing practice — keep using the same file skills in every unit.

Evaluation & Expectations

  • Be respectful, stay on task, and follow digital citizenship guidelines.
  • Keep files organized in your H:/ICT_9 folder.
  • Submit on time to the correct MS Teams assignment.
  • Typing: aim for steady improvement in accuracy and WPM.

Daily Typing Routine

  • Start of class: 10-15 minutes of typing warm-up.
  • Track progress; we’ll compare baseline vs. later in the term.
  • Use headphones when available.

Digital Literacy Fridays

Short, practical mini-lessons on Fridays. Topics include: passwords, online identity & footprint, evaluating sources, fair use & attribution, file hygiene, and responsible AI use.
