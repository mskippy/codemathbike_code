Activity D ‚Äî Download & Extract a ZIP Folder (Game Assets)

Objective: Download a ZIP file from a website, extract it, move it to the correct folder, and document the process.

  1. Go to the download page
    Open Microsoft Edge and go to:
    https://kenney.nl/assets/space-shooter-redux
  2. Download the ZIP file
    • Scroll down and click the Download button.
    • When Edge asks what to do, click Save.
    • The file will download to your Downloads folder.
  3. Find the ZIP file
    • Open File Explorer.
    • Click Downloads on the left.
    • Look for space-shooter-redux.zip.
  4. Extract the ZIP file
    • Right-click on space-shooter-redux.zip.
    • Click Extract All‚Ä¶
    • A dialog box will appear asking where to extract the files.
    • Do not change anything ‚Äî just click Extract.
  5. Locate the extracted folder
    • A new folder called space-shooter-redux will appear.
    • This is the folder you will use (not the ZIP file).
  6. Move the folder to your Game Development folder
    Move the space-shooter-redux folder into:
    ICT 8 ‚Üí Game Development
    Ways to move the folder
    • Option A: Click & Drag
      Click and hold the folder, drag it to your Game Development folder, then release.
    • Option B: Right-click ‚Üí Cut / Paste
      Right-click the folder ‚Üí Cut
      Open your Game Development folder ‚Üí right-click ‚Üí Paste
    • Option C: Keyboard shortcuts
      Click the folder once ‚Üí press Ctrl + X
      Open your Game Development folder ‚Üí press Ctrl + V
  7. Take a screenshot
    • Take a screenshot showing the space-shooter-redux folder inside your Game Development folder.
    • Add this screenshot to your 1.2 Essential Skills Webquest.
What NOT to do (common mistakes)
  • ‚ùå Do not work directly out of the .zip file.
  • ‚ùå Do not move the ZIP file instead of the extracted folder.
  • ‚ùå Do not change the extract location unless instructed.
  • ‚ùå Do not delete the folder after extracting it.

Final check: the folder should NOT end in .zip.
