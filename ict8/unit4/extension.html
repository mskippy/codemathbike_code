Path A — Make a Second Page + Site Navigation

  1. Duplicate your current file and rename it about_<firstname><lastinitial>.html<> in H: → ICT8 → 4. Web Development.
  2. Change the <title> and top heading to About. Write a short paragraph about your project or hobbies.
  3. Add simple site navigation to both pages: 
    <nav>
  <a href="/H/ICT8/4. Web Development/firstname_lastinitial.html">Home</a>
  <a href="/H/ICT8/4. Web Development/about_firstnameL.html">About</a>
</nav>
    Adjust the H: paths to match your files.
  4. Style the nav: add spacing and a hover effect in your <style> block.

Success check

  • Both links work (Home ↔ About).
  • Titles/headings are clear and different per page.

Path B — Add a Downloadable Document

Create a small Word document (1–2 paragraphs) and link it for download from your site.

  1. Create a Word file named Unit4_Plan_<firstname><lastinitial>.docx<> and save it in H: → ICT8 → 4. Web Development.
  2. Add a download link to your page (under a new heading like “My Plan”): 
    <a href="/H/ICT8/4. Web Development/Unit4_Plan_firstnameL.docx" download>Download my plan (.docx)</a>
    If the browser opens it instead of downloading, right‑click → Save link as.
  3. Below the link, add a short sentence explaining what’s inside the file.

Success check

  • Link points to the correct .docx<> in your H: drive folder.
  • Text explains the file contents.

Path C — Embed a YouTube Video (Responsive)

  1. Pick a school‑appropriate YouTube video that relates to your interests.
  2. Copy the video ID (the letters after v=<> in the URL).
  3. Add this block to your page where you want the video: 
    <div class="embed video" aria-label="YouTube video embed">
  <iframe 
    src="https://www.youtube.com/embed/VIDEO_ID" 
    title="YouTube video player" allowfullscreen
    referrerpolicy="strict-origin-when-cross-origin"></iframe>
</div>
    Replace VIDEO_ID<>.
  4. Add a one‑sentence caption under the video using <figcaption> or a plain paragraph.

Success check

  • Video scales on narrow and wide screens.
  • Caption explains why you picked it.

Bonus — Image Gallery with Credits

Create a 2‑column mini‑gallery using <figure> + <figcaption>. Use images with proper credits.

<section aria-label="Gallery" class="gallery">
  <figure>
    <img src="FULL_IMAGE_URL" alt="Short description">
    <figcaption>Title — credit/source</figcaption>
  </figure>
  <figure>…</figure>
</section>

Add a little CSS (padding/border‑radius/shadow) to make it tidy.

Stretch Ideas (pick any)

  • Create jump links (table of contents at the top linking to sections using id="...").
  • Add a mailto contact link: <a href="mailto:firstnameL@school.example">Email me</a><>.
  • Move your inline CSS into a separate file custom.css<> and include it with <link rel="stylesheet" href="/H/ICT8/4. Web Development/custom.css"><>.
  • Use a semantic layout: wrap content in <header>, <main>, <footer>.

Where should I save my files?

Use H: → ICT8 → 4. Web Development. Keep filenames simple (letters, numbers, underscores). Avoid spaces when possible.

Goals

  • Link between pages using proper href paths.
  • Offer a file download using the download attribute.
  • Embed third‑party media responsively with <iframe>.

Check List

  • Clear headings; tidy spacing.
  • Working links (no 404s or blanks).
  • Alt text on images; short captions/credits.
  • No personal info beyond first name + last initial.

Troubleshooting

  • Link opens wrong file: Double‑check your path and filename.
  • Video doesn’t show: Ensure the VIDEO_ID<> is correct; some videos block embedding.
  • Download doesn’t download: Try right‑click → Save link as.
