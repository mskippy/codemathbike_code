Unit 2 — Digital Media

In this unit, you’ll learn how to use Canva and Excel to create digital work that is organized, clear, and visually appealing.

What this unit is about

Digital media is everywhere — in school, at home, and online. This unit focuses on a few core tools and skills so you can build projects you’re proud of without feeling overwhelmed.

What you’ll learn

  • Basics of design principles (contrast, alignment, spacing, hierarchy)
  • How to create clean, readable layouts in Canva
  • How to build tables, use simple formulas, and make charts in Excel
  • How to present information visually so it’s easy to understand
  • How to reflect on your work and explain your design or data choices
