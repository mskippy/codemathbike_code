Overview

Big idea: Build a set of clear “How-To” posters that help you (and others) use school technology confidently and independently.

In this lesson, you will create 10 How-To posters, each focused on one essential tech skill. You will practise the skills, take screenshots while you perform them, and turn your steps into easy-to-follow instructions.

By the end of this lesson, you will be able to:

  • Save files correctly to your H: Drive and OneDrive.
  • Move and rename files and folders to stay organized.
  • Extract .zip files and understand why they are used.
  • Export files as PDF and PNG from common apps.
  • Upload assignments properly to Teams.
  • Use the Snipping Tool to take and edit screenshots.
  • Use split screen / Snap Layouts to work more efficiently.

You should have completed: the earlier Unit 2 lessons on file management, saving, and screenshots.

The 10 Essential How-To Posters

You will create 10 separate posters, one for each topic below. Click each heading to see the suggested steps and ideas.

1. How to Save to Your OneDrive

Your OneDrive is your personal cloud storage space. Files saved here are accessible from any device.

  1. Open a document in Word, PowerPoint, or Excel.
  2. Go to File → Save As.
  3. Double click on OneDrive.
  4. Pick or create a folder (for example, ICT8<>).
  5. Give the file a clear name (for example, Lastname_ProjectName<>).
  6. Click Save.

Poster tip: Include screenshots showing where the H: Drive appears in the Save As window.

2. How to Save to Your H: Drive

Your H: Drive is your personal storage space on the school network. Files saved here are accessible only from school computers.

  1. Use File Explorer to navigate to your H: Drive.
  2. Go to File → Save As.
  3. Double click on This PC.
  4. Choose your H: Drive from the list.
  5. Pick or create a folder (for example, ICT8<>).
  6. Give the file a clear name (for example, Lastname_ProjectName<>).
  7. Click Save.
3. When to Save to OneDrive vs. H: Drive

Use OneDrive for cloud-based Office files and H: Drive for downloaded files and project assets.

  • OneDrive: Word, Excel, PowerPoint you want to access from home; autosave enabled.
  • H: Drive: Canva downloads, images, audio, video, game assets, and large project files.
  1. In Word/PowerPoint/Excel, click your account name and choose OneDrive to save to the cloud.
  2. Use File → Save As → OneDrive for cloud storage.
  3. Use File → Save As → This PC → H: Drive for local school storage.

Poster tip: Show side-by-side screenshots of saving to OneDrive and saving to H: Drive.

4. How to Extract a ZIP File

You’ll practise extracting a ZIP file using a real game asset pack from Kenney.

  1. Open your web browser (Edge or Chrome).
  2. Go to: https://kenney.nl/assets/space-shooter-redux
  3. Click the download button for the asset pack (look for Download or Direct download).
  4. When the download finishes, open File Explorer and go to your Downloads folder.
  5. Find the file that ends with .zip (for example, space-shooter-redux.zip<>).
  6. Right-click the ZIP file and choose Extract All....
  7. Leave the default folder location (your Downloads folder) and click Extract.
  8. After extraction, open the new folder (for example, space-shooter-redux<>) and check that you can see the images/files inside.
  9. Finally, move the extracted folder from Downloads to a folder on your H: Drive (for example, H:\ICT8\3. Game Development<>).

Poster tip: Show screenshots of the Kenney page, the ZIP file in Downloads, the “Extract All...” window, and the final folder on your H: Drive.

5. How to Move Files & Folders

Moving files keeps your work organized and makes it easier to find important project folders later.

In this example, you’ll move the space-shooter-redux folder you extracted from your Downloads folder to your H: Drive.

  1. Open File Explorer.
  2. In the left sidebar, click on Downloads.
  3. Find the folder named something like space-shooter-redux.
  4. Click once on the folder to select it.
  5. Use one of these methods to move it:
    • Drag and drop: drag the folder from Downloads to your H: Drive in the left sidebar.
    • Cut and paste: right-click the folder → Cut, then go to H: Drive, open your ICT8<> or Game Assets<> folder, right-click in the empty space → Paste.
  6. Open the folder in its new location on your H: Drive to confirm it moved successfully.

Poster tip: Show a screenshot of the folder in Downloads and another of the same folder safely stored on your H: Drive.

6. How to Rename Files Correctly

Good file names make it easier for you (and your teacher) to find your work.

  1. Right-click the file name in File Explorer.
  2. Click Rename.
  3. Type a clear name using this pattern:
    • AssignmentName<>
    • Make sure it is in the correct subject folder
    • Use numbers to keep work in order
  4. Press Enter to save the new name.

Poster tip: Include examples of good and bad file names.

7. How to Take a Screenshot (Snipping Tool)

Use the Snipping Tool to capture only the parts of the screen you need.

  1. Click the Start menu and type Snipping Tool, then open it.
  2. Click New to start a snip.
  3. Drag to select the area you want to capture.
  4. Use the pen or highlighter to mark important parts (optional).
  5. Click Save (disk icon), choose your H: Drive and folder.
  6. Give the screenshot a clear name and click Save.

Poster tip: Show one screenshot before markup and one with circles/boxes added.

8. How to Save a File as a PDF

PDFs are useful for sharing and printing because they look the same on any device.

  1. Open your document in Word, PowerPoint, or Canva.
  2. Go to File → Save As (or Download in Canva).
  3. Choose the file type PDF.
  4. Select your save location (usually your H: Drive).
  5. Give the PDF a clear name and save it.

Poster tip: Highlight the file type dropdown where you choose PDF.

9. How to Export a PNG from Canva

PNG files are high-quality image files, perfect for posters and websites.

  1. Finish your design in Canva.
  2. Click Share → Download.
  3. In the file type menu, choose PNG.
  4. Click Download.
  5. When prompted by the browser, choose your H: Drive and a folder.

Poster tip: Show the Canva download panel with PNG selected.

10. How to Upload a File to Teams

Uploading correctly ensures your teacher can see and mark your work.

  1. Open Teams and go to your class.
  2. Click on the Assignments tab.
  3. Select the correct assignment.
  4. Click Add work.
  5. Choose Upload from this device.
  6. Find your file on your H: Drive and click Open.
  7. Wait for it to upload, then click Turn in.

Poster tip: Include a screenshot of the “Add work” and “Turn in” buttons.

Tips & Tricks for Creating Great Posters

Use these tips to make your posters clear, accurate, and easy to follow. Click each heading to expand.

1. Practise the Skill First

Do the full process once or twice before you start making your poster. This helps you understand which steps are most important and in what order.

2. Take Screenshots While You Work

Keep the Snipping Tool open while you perform the skill. Capture each important step as you go so your screenshots match your instructions.

3. Use Markup to Highlight Key Areas

Use the Snipping Tool’s pen and highlighter tools to draw boxes, circles, or underlines around important buttons, menus, and icons.

4. Keep Formatting Consistent

Use the same fonts, sizes, colours, and step formatting on every poster so your set feels like a matching collection.

5. Use Split Screen While You Build

Put your instructions on one side and your poster app on the other. This makes it easier to follow each step and check your screenshots.

6. Stay Organized

Create a folder (for example, H:\ICT8\How-To Posters<>) and save all screenshots and posters there so they are easy to find when you upload.

7. Review Before You Hand In

Check that each step matches the screenshot, that your highlights are clear, and that your poster is readable and not too cluttered.
