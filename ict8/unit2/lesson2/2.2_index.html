Overview

Big idea: Use Excel to work with small, realistic datasets so you can organize information, use formulas, and create charts that tell a story.

In this lesson you will:

  • Create simple tables with clear headings.
  • Enter and edit text and numbers correctly.
  • Use basic formulas (sum, area, perimeter, squares).
  • Use AutoSum and AutoFill to work efficiently.
  • Sort data alphabetically and by number.
  • Create and label basic charts (column/bar).
  • Explain what your chart shows in your own words.

What kind of triangle is it? - Extension

In this task you will use Excel and math to determine the different types of triangles.

The triangle rule (for c as the longest side)

  • If a^2 + b^2 > c^2<> → the triangle is acute.
  • If a^2 + b^2 = c^2<> → the triangle is right.
  • If a^2 + b^2 < c^2<> → the triangle is obtuse.
