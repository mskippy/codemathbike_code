Intro

Every game needs a background to set the scene and make the world feel complete. In this step, you will add a background image to your project so your top-down shooter doesn’t just appear on a blank screen. The background helps create the atmosphere and makes your game more engaging.

Add a Background Object

In the Objects panel, click Add a new object → Tiled Sprite. Name it Background.

Select the Background Image

Click Choose image and select your background from the assets folder. Press Open then Apply to add the object

Place It in the Scene

Open the Scene Editor and drag Background into the scene.

Cover the Game Window

Use the resize handles so the background fills the entire game window (no white edges).

Lock the Background

In the Properties panel, click on the Lock icon to keep the background in place..

Take a screenshot of your layout and save it in your folder.

Adding an animation

Now that we have a background, we need it to look like we are in space and our spaceship is moving.

To do this we are going to add an action to the background by adding events

Adding events

Events are how we tell the objects in our game what to do. Click on your Level 1 (Events) tab and press Add an event.

Add action

Now we need to add an action to the event. Click on Add action.

Now we need to define the action. The action we want is to offset the image background.

Click on the Background → Image X offset and change the Modification sign to +(add) and the Value to 1. Finish by pressing OK.

Save your work and press Preview to see your background move!

Take a screenshot of your event sheet and save it in your folder.
