Overview

Big idea: Your website structure is finished and your content is in place. Now it’s time to make your website look the way you planned.

In this lesson, you will format your website by editing the style.css file from your Layout Kit so your site matches the visual design you created in 5.2 (your Canva mock-up).

Everyone started with the same template. By the end of this lesson, your site should feel like your design.

Reminder: The final checklist happens in 5.6. Today is for formatting + polishing.

Start Here: What You’ll Probably Want to Change

Most students begin with the same formatting goals. You do not need to do everything — choose changes that support your design plan.

  • Fonts & text (style, size, spacing)
  • Colours (backgrounds, text, accents)
  • Spacing (tight vs open layout)
  • Cards & sections (borders, rounding, shadows)
  • Images (adding images, placing text beside images, image backgrounds)

All of these changes happen in one file: style.css

Before You Edit

Rule for today: Edit values, not structure.

✅ Safe to edit

  • Colours: color, background-color
  • Text: font-family, font-size, line-height
  • Spacing: margin, padding
  • Cards: border, border-radius, box-shadow
  • Hover effects (optional): :hover

⛔ Do not change (yet)

  • Layout rules that control navigation placement
  • Large sections of code you don’t understand
  • Selector names (the left side of CSS rules)
  • Deleting blocks of CSS “to clean it up”

Best workflow: one change at a time → save → refresh → observe. If something breaks, undo your last change.

Step 1 — Open Your Canva Mock-Up

Open the Canva design you created in 5.2. This is your guide. You are not guessing — you are translating a plan into code.

  • What fonts did you choose?
  • What colours set the mood?
  • Is the layout tight or airy?
  • Do sections feel like cards, panels, or full-width blocks?

Step 2 — Open Your Stylesheet

  1. Open your project folder in VS Code.
  2. Open style.css.
  3. Read the comment block at the top of the file.

Quick check: If you edit style.css and nothing changes when you refresh, your HTML pages may not be linked to it.

If your CSS isn’t loading (click to expand)
  • Confirm style.css is in the same folder as your HTML files.
  • Confirm each HTML file has: <link rel="stylesheet" href="style.css">
  • Hard refresh your browser (Ctrl + Shift + R / Cmd + Shift + R).

Change 1 — Fonts & Text (Typography)

Typography has the biggest impact on how your site feels. Start here.

What you can change

  • Font style (font-family)
  • Text size (font-size)
  • Line spacing (line-height)
  • Heading vs body contrast (bigger headings, readable body text)

Customization ideas

  • Clean + minimal: simple sans-serif fonts and larger spacing
  • Bold + energetic: strong headings and tighter layout
  • Creative vibe: expressive headings with readable body text

Reference: W3Schools – CSS Fonts

Change 2 — Colours & Mood

Colour sets the tone of your website. Match your Canva palette as closely as you can.

What you can change

  • Page backgrounds
  • Text colour
  • Navigation colour
  • Accent colour (used consistently)

Tip: If text is hard to read, simplify your colours and increase contrast.

Reference: W3Schools – CSS Colors

Change 3 — Spacing & Layout Feel

Spacing controls whether your site feels calm or cluttered.

What you can change

  • Space inside sections (padding)
  • Space between sections (margin)

Reference: W3Schools – CSS Box Model

Change 4 — Cards & Sections

Cards and sections help organize content. Small changes here can make your site feel much more polished.

Common upgrades

  • Rounded corners (border-radius)
  • Soft borders (border)
  • Subtle depth (box-shadow)
  • Consistent spacing inside sections (padding)

Reference: W3Schools – CSS Borders

Change 5 — Images (Common Layout Goals)

Images add personality and visual interest. Use them intentionally so they support your design plan.

Goal A: Add an image with text beside it

Use Flexbox to place an image and text side-by-side inside a section.

Reference: W3Schools – CSS Flexbox

Helpful terms: display: flex, align-items, gap

Goal B: Make images look cleaner

Control image sizing, spacing, and corners so images feel consistent across your site.

Reference: W3Schools – CSS Images

Goal C: Use an image as a background

Background images work well for header sections or featured areas.

Reference: W3Schools – CSS Backgrounds

Helpful terms: background-image, background-size, background-position

Optional Polish — Subtle Effects

Small effects can add polish if used carefully (and not overdone).

  • Hover effects on navigation links
  • Hover effects on cards
  • Small transitions (optional)

Reference: W3Schools – CSS Pseudo-classes

How to Use Reference Guides Without Getting Lost

  1. Decide what you want to change (font, colour, spacing, cards, images).
  2. Open the matching W3Schools page (links above).
  3. Copy only the line(s) you need.
  4. Paste into style.css and adjust values to match your design.
  5. Save → refresh → check at least 2 pages.

You are adapting examples — not copying full pages of code.
